ഈ അധ്യായതിന്റെ തുടക്കത്തിൽ നമ്മൾ സ്ട്രക്ചർ ഡാറ്റാ ഇനത്തെകുറിച്ചും അതിന്റെ ഉപയോഗത്തേക്കുറിച്ചും ചർച്ച ചെയ്തു. ഈ ഭാഗത്ത്‌ നമ്മൾ സ്ട്രക്ചർ വാരിയബിളിനെ പോയിന്ററിന്റെ സഹായത്താൽ എങ്ങനെ ഉപയോഗിക്കാമെന്ന് പഠിക്കുന്നു.

The syntax for accessing the elements of a structure using pointer is as follows:

Structure_pointer->element_name

Examples:

Struct employee
{
into ecode;
char ename[15];
float salary;
} *entr;

eptr->ecode = 657346;
cin.getline (eptr->ename,15);
cin>>eptr->salary;
cout<<eptr->salary * 0.12;

Self Referential Strucuture

ഒരു structure ലെ ഒരു അംഗത്തെ അതേ structure ഉള്ളിൽ തന്നെ പോയിന്ററായി പ്രഖ്യാപിക്കുമ്പോൾ, strucure നെ self referential എന്ന് വിളിക്കുന്നു

Example:

struct employee
{
int ecode;
char ename[15];
float salary;
employee *ep;
};
