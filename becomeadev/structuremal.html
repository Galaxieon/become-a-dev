പരസ്പര ബന്ധമുള്ള ഡാറ്റ ഇനങ്ങളുടെ ശേഖരത്തെ പ്രതിനിധീകരിക്കുന്ന user defined ഡാറ്റ തരമാണ് Structure

Defining a structure

ഒരു structure declare ചെയ്യാൻ struct statement ഉപയോഗിക്കുന്നു

struct [struct tag]
{
Member definition;
Member definition;
......................................
......................................
};

ഒരു structure ലെ അംഗങ്ങളെ structure members എന്നും വിളിക്കുന്നു

For Example

struct student
{
int rollno;
char grade;
float percentage;
};

Accessing elements of a structure

ഒരു structure ലെ ഘടകങ്ങൾ ഡോട്ട് (.) ഓപ്പറേറ്റർ അല്ലെങ്കിൽ period operator ഉപയോഗിച്ച് access ചെയ്യുന്നു.

The syntax for accessing elements is

[Object name][Operator][Member variable Name]

For Example

s.name.
#include<iostream>
using namespace std;
struct student
{
int rollno;
int m1,m2,m3;
};
int main()
{
student s; //Object declared
cout<<"Enter the marks in Three subjects";
cin>>s.m1>>s.m2>>s.m3;
cout<<"Total="<<s.m1+s.m2+s.m3;
return 0;
}

Nested Structure

മറ്റൊരു structure യ്ക്കുള്ളിൽ സ്ഥാപിച്ചിരിക്കുന്ന ഒരു structure നെ nested structure എന്ന് വിളിക്കുന്നു

Definition A Definition B
struct date
{
Short day;
Short month;
Short year;
};
struct student
{
into adm_no;
char name [20];
date dt_adm;
float fee;
}; 		Struct student
{
int adm_no;
char name [20];
struct date
{
Short day;
short month;
Short year;
} dt_adm;
float fee;
};

Elements of nested structure are accessed as follows:

student st;
cin>>st.dt_adm.day
>>st.dt_adm.month
>>st.dt_adm.year;

Difference between Array and Structure

Array Structure
സമാന ഡാറ്റയുടെ ഒരു ശേഖരമാണ് ഒരു array (Homogeneous) വ്യത്യസ്ത ഡാറ്റയുടെ ശേഖരമാണ് structure (Heterogeneous)
ഒരു array യിലെ ഘടകങ്ങൾ access ചെയ്യാൻ ഒരു array index ഉപയോഗിക്കുന്നു ഒരു structure ലെ ഘടകങ്ങൾ dot (.) ഓപ്പറേറ്റർ ഉപയോഗിച്ച് access ചെയ്യുന്നു
Array is derived data type Structure is user defined data type
