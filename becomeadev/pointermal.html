കമ്പ്യൂട്ടറിൽ ഡാറ്റ ഒരു memory cell ൽ store ചെയ്യുന്നു. ഓരോ സെല്ലിനും ഒരു unique address ഉണ്ട് . pointer variable memory location ന്റെ വിലാസം സൂക്ഷിക്കുന്നു.

Declaring a pointer

The syntax is

Data_Type *Pointer_Variable;

For Example

int *ptr;

ഇവിടെ ptr ഒരു പോയിന്റർ വേരിയബിളാണ്

Initializing a pointer. We can initialize a pointer as,

Ptr=&a;

ഇവിടെ pointer variable ptr ൽ വേരിയബിൾ a യുടെ address അടങ്ങിയിരിക്കുന്നു.
ഒരു പോയിന്റർ വേരിയബിൾ സംഭരിച്ചിരിക്കുന്ന value നേടുന്നതിന് * operator ഉപയോഗിക്കുന്നു.
address of operator(&) ഒരു വേരിയബിളിന്റെ address വീണ്ടെടുക്കാൻ ഉപയോഗിക്കുന്നു

Example

#include<iostream>
using namespace std;
int main()
{
int a=10;
int *Ptr=&a; cout<<"The value of a="<<a<<"\n";
cout<<"The address of a="<<Ptr;
return 0;
}

Output:
The value of a=10
The address of a= oxffoo
